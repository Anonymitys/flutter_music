# flutter_music

爬取QQ音乐网页端数据，基于flutter写的一个在线音乐播放器，实现了QQ音乐首页，官方歌单，推荐歌单，新歌新碟，歌手，歌手详情，排行榜，MV排行榜和MV速递，支持非会员和付费歌曲的播放功能
